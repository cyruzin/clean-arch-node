# Clean Architecture in Node

Clean Architecture in Node.

## Running

Rename the **.env_example** to **.env** and fill the variables.

Make sure you have Docker installed in your machine. This app uses Postgres.

```sh
 docker-compose up -d
```

Install dependencies:

```sh
 yarn install
```

Then:

```sh
 yarn start
```

## Tests

```sh
 yarn test
```
